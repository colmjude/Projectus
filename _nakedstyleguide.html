Horizontal rule

The hr element represents a paragraph-level thematic break, e.g. a scene change in a story, or a transition to another topic within a section of a reference book.

Quotes and Citations

Blockquotes

The blockquote element represents a section that is being quoted from another source.

Paul Robert Lloyd, in his barebones project. 9th May 2012

Additionally, you might wish to cite the source, as in the above example. The correct method involves including the cite attribute on the blockquote element, but since no browser makes any use of that information, it’s useful to link to the source also.

Citations

The cite element is used to represent the title of a work (e.g. a book, essay, poem, song, film, TV show, sculpture, painting, musical, exhibition, etc). This can be a work that is being quoted or referenced in detail (i.e. a citation), or it can just be a work that is mentioned in passing. Example:

Universal Declaration of Human Rights, United Nations, December 1948. Adopted by General Assembly resolution 217 A (III).

Inline quotes

The q element is used for quoting text inline. Can be nested.

I once said, Your site is only as good as the experience one has with it. I still believe that.

Lists

Ordered list

The ol element denotes an ordered list — numbers (or letters/numerals) not bullets. Individual list items require surrounding <li> tags.

  1. This is an ordered list.
  2. This is the second item, avec sub list
    1. This is the sub list — also ordered.
    2. It has two items.
  3. This is the final item.

Unordered list

The ul element is used for an unordered list — bulleted list not numbered. Same as ordered lists individual items are wrapped with <li> tags.

  • United Kingdom of Great Britain and Northern Ireland:
    • England
    • Scotland
    • Wales
    • Northern Ireland
  • Republic of Ireland
  • Isle of Man
  • Channel Islands:
    • Bailiwick of Guernsey

    • Block level elements, such as paragraphs can be contained in list items.

Definition list

dl
The dl element is for another type of list called a definition list. The content of a dl consists of dt and dd pairs.
dt
Definition Term (dt), what is to be defined. Or the parent element.
dd
Definition description (dd), description of what is in the dt.
Some dt
Another dt
dt terms may stand on their own without an accompanying dd, but in that case they share descriptions with the next available dt. You may not have a dd without a parent dt.

Text-level Semantics

There are a number of inline HTML elements you may use anywhere within other elements.

Links and anchors

This is a link to my blog and this is a link to the top of the page. Both are marked up using an a element.

Stressed emphasis

Using the i tag indicates that the user should read the text in an alternate voice or mood. It doesn’t imply any importance or emphasis.

For additional emphasis use the em tag.

Strong importance

Use the b tag for text that is intended to be stylistically offset from the normal prose without conveying any extra importance.

If the text has extra importance then use the strong tag.

Small print

This is the small print, it is marked up using a small tag and is used for legalese, or terms and conditions.

Definition

The dfn element is used to highlight the first use of a term. The title attribute is used to describe the term.

Abbreviation

The abbr element is used for any abbreviations and acronyms, e.g. HTML.

Time

I lasted edited this section on Thursday 10th May using the time element to remember that fact.

Variable

The var element is used for mathematical expressions or placeholder text. 2X = 8.

Sample output

Use the samp element to represent (sample) output from a program or computing system.

E.g. Status: 200 OK

Keyboard entry

The kbd element is used to denote user input.

E.g Konami Code ↑↑↓↓←←→→BA

Superscript and subscript text

Use sup for superscript text such as 1st and use sub for subscript text such as x10.

Marked or highlighted text

The mark element doesn’t attach any importance to the content within it, other than to show that it’s currently of interest.
